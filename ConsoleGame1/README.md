# Farming Simulator

> 콘솔 프로젝트로 제작한 농장 시뮬레이터 입니다

## 프로젝트 정보
- 게임 장르 : 시뮬레이션
- 제작 인원 : 1 인 (개인 프로젝트)
- 제작 기간 : 2일
- 사용 엔진과 언어 : Visual Studio 2022, C#

## 플레이 방법
1. 방향키로 캐릭터를 움직이세요
2. z,x,c키로 밭(네모)을 클릭하여 성장시키고 채집합니다.
