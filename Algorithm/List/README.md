# List
- 런타임 중 크기를 확장할 수 있는 배열기반의 자료구조
-  배열요소의 갯수를 특정할 수 없는 경우 사용이 용이

원하는 기능 여기있나 보기
https://learn.microsoft.com/ko-kr/dotnet/api/system.collections.generic?view=net-8.0

```cs
//정적배열 - 값을 안써도 미리 5자리 차지함
string[] strings = new string[5];

strings[0] = "안";
strings[1] = "녕";

//동적배열 - 상황에 따라 하나씩늘려줌
List<string> list = new List<string>();
list.Add("안");
list.Add("녕");
```

<순차리스트 시간복잡도>
접근  탐색  삽입  삭제
O(1)  O(n) O(1)  O(n)
```cs

static void Main(string[] args)
{
	List<string> list = new List<string>();

	//삽입
	list.Add("안"); //O(1) 바로넣으니까 빠름
	list.Add("녕");
	list.Add("하");
	list.Add("세");

	list.Add("요");
	list.Insert(2, "집어넣을거"); //O(n) 나머지 원소들을 미니까 좀 느림
	list.Insert(0, "맨앞에");

	//삭제
	list.Remove("하"); //O(n) 자리 삭제되면 땅겨짐, 중복 5개있을때 맨앞에 하나만 지워짐
	list.RemoveAt(2); //최선:O(1) 맨 뒤에꺼 지울때, 평균:O(n), 최악:O(n) => O(n)
	//list.Clear();

	//접근
	list[3] = "바꾼 데이터"; //O(1) 바로 값 바꾸니까
	string text = list[2];

	//탐색
	int index = list.IndexOf("!"); //O(n) 얘가 몇번째인지
	bool contain = list.Contains("!"); //있는지 없는지
}
```

리스트는 일부로 여유있게 크게 만들어놓고 몇개를 쓰고있는지 체크하는 구조
```cs
List<string> list = new List<string>();

list.Count; //지금 몇개쓰고있는지
list.Capacity; //용량이 얼마난지
```
